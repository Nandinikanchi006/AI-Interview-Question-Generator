# 🎤 AI Interview Question Generator Pro

An AI-powered interview preparation platform that generates personalized interview questions from your resume and projects using Google's Gemini AI.

---

## 🚀 Features

### 📄 Resume Analysis
- Upload your resume in PDF format
- Extracts text automatically
- Displays resume preview

### 🎯 AI Interview Questions
- Generates personalized interview questions
- Questions based on:
  - Resume content
  - Selected role
  - Difficulty level

### 📂 Project Question Generator
Generate project-specific interview questions based on:
- Project Name
- Project Description

Questions include:
- Technical Questions
- Design Questions
- Deployment Questions
- Challenges Faced
- Future Improvements

### ✍️ AI Answer Evaluation
Evaluate your answers using Gemini AI.

Provides feedback on:
- Technical Accuracy
- Communication Skills
- Confidence Level
- Suggestions for Improvement

### 📊 Analytics Dashboard
Visualize interview question distribution using Plotly charts.

Tracks:
- Technical Questions
- HR Questions
- Project Questions

### 🎨 Modern UI
- Animated Gradient Background
- Dark Theme
- Interactive Tabs
- Professional Dashboard
